where 테이블 조건 --- 2

```sql
0. select : 조회
1. from : 타겟 테이블 
2. where: 주어진 조건에 맞는 데이터 추출
3. join : 테이블 연결
4. on : 연결할 테이블
5. group by: 공통적인 데이터끼리 묶어 그룹 만들기
6. having: 주어진 조건에 맞는 그룹 추출
7. order by: 데이터 정렬
    a. asc: 기본값, 오름차순
    b. desc: 내림차순
8. limit: 상위 n개 레코드 추출
```
